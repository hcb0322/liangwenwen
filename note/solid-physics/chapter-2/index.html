第二章 晶体的结合

原子的电负性

  • 使原子失去一个电子所需的能量称为原子的电离能。
  • 一个中性原子获得一个电子成为负离子所释放出的能量叫电子亲和能。
  • 原子电负性=0.18（电离能+亲和能）

晶体的结合类型

  • 共价结合
  • 离子结合
  • 金属结合
  • 分子结合
  • 氢键结合

结合力及结合能

  • 势能：$u(r)=a\frac{A}{r^m}+\frac{B}{r^n}$
  • 互作用力：$f(r)=-\frac{du}{dr}=-(\frac{mA}{r^{m+1}}-\frac{nB}{r^{n+1}})$

分子力结合

  • 极性分子结合
    对于全同的极性分子
    $u(r)=-\frac{p^2}{2\pi\varepsilon_0r^3}\ \ \ \ \ p=ql\ ,\ l_1=l_2=l$
  • 极性分子与非极性分子的结合
    $u(r)=-\frac{\alpha p_1^2}{4\pi^2\varepsilon_0^2r^6}$

共价结合

  • 两原子中自旋相反的价电子可以为两原子共享，使得体系能量最低，自旋相反的两电子称配对电子，称配对的电子结构为共价键。

离子结合

  • 埃夫琴提出计算马德隆常数，巴晶体看成是由埃夫琴晶胞来构成，埃夫琴晶胞内所有离子的电荷代数和为 0 ，把这些中性晶胞对参考离子的库仑能量的贡献份额加起来就得到了马德隆常数。
  • $\mu=\underset{j}{\sum}’\pm \frac{1}{a_j}$  同号 $-$  异号 $+$
