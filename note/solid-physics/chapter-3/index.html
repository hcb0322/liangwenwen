三维晶格的振动

  • 声学波有 $3$ 支，三维晶格的波矢密度为 $\frac{V_c}{(2\pi)^3}$ ，光学波有 $3n-3$ 支

简正振动、声子

晶格振动能、声子

  • 频率为 $w_i$ 的谐振子的振动能为：$\varepsilon_i=(n_i+\frac{1}{2})\hbar w_i$
  • 晶格振动能：$\underset{i=1}{\overset{3N}{\sum}}(n_i+\frac{1}{2}\hbar w_i) \ \ n_i=0,1,2 \ …$
  • 晶格的振动能时量子化的，能量的增减是以 $\hbar w$ 为计量，赋予 $\hbar w$ 一个假想的携带者——声子，即声子是一个晶格振动的能量的量子
    • 声子是准粒子，$\hbar q$ 为声子的准动量。
    • 声子不携带物理动量
    • 声子的另一个性质是声子的等价性
  • 晶体的振动能高取决于：
    • 声子数目多
    • 能量大的声子数目多
  • 平均声子数目（利用玻尔兹曼统计理论）$n(w)=\frac{1}{e^{\frac{\hbar w}{k_BT}}-1}$
    当 $T=0K$ 时，$n(w)=0$，即 $T>0K$ 时才有声子
    当温度很高时，$n(w)\approx\frac{k_BT}{\hbar w}$

晶格振动热容理论

热容理论

  • 模式密度 $D(w)$ ：单位频率区间的格波振动模式数目
    $D(w)=\frac{dz}{dw}$
  • 在一维中：
    $dz=\frac{L}{2\pi}dq\times2 \ \ w=vq \ \ dw=vdq$
    $D(w)=\frac{dz}{dw}=\frac{L}{\pi}\cdot\frac{1}{dw/dq}=\frac{L}{\pi v}$
  • 总的热振动能为：
    $E=\int^{w_m}_0\frac{\hbar wD(w)}{exp(\frac{\hbar w}{k_BT})-1}$
    $C_v=(\frac{\partial E}{\partial T})=\int^{w_m}_0k_B(\frac{\hbar w}{k_BT})^2\frac{exp(\frac{\hbar w}{k_BT})D(w)}{exp(\frac{\hbar w}{k_BT})-1}$

爱因斯坦模型

  • 高温时： $C_v=3Nk_B$ （三维）
  • 低温时：爱因斯坦热容比 $T^3$ 更快地趋于 $0$ ，这与实验偏差很大。
  • 原因：爱因斯坦把所有格波都视为光学波，没有考虑长声学波在甚低温时对热容的主要贡献。

德拜模型

  • 把格波作为弹性波处理（长声学波。

  • 在一维中：
    $\int^{w_m}_0D(w)=N$
    $w_m=\frac{N\pi v}{L}= w_D$

    $\Theta_D=\frac{\hbar w_B}{k_BT}$

  • 变量变换：
    $x=\frac{\hbar{w_B}}{k_BT}=\Theta_D$
    $C_v=\frac{LK^2_BT}{\pi v\hbar}\int^{\Theta_D}_0\frac{x^2e^x}{(e^x-1)^2}dx$

  • 温度较高时:
    $\frac{x^2e^x}{(e^x-1)^2}\approx1$
    $C_v=Nk_B$

  • 低温时：
    $\int^{\Theta_D}_0\frac{x^2e^x}{(e^x-1)^2}dx=\frac{\pi^2}{3}$
    $C_v=\frac{\pi^2k_BN}{3}(\frac{T}{\Theta_D})$
